# Expense Tracking System

Expense Tracking System ek React.js based frontend application hai jo users ko apne daily expenses manage karne mein help karta hai.

## Key Features
- Expense add, view aur manage kar sakte hain
- Data browser ke Local Storage mein save hota hai
- Koi backend ya server dependency nahi
- Simple aur clean UI

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS

## Note
Ye project purely frontend hai aur data user ke browser mein locally store hota hai.
